# Full-Stack Boilerplate

This is a simple boilerplate for back-end(server) and front-end(web) using PNPM.

## Back-end

- ExpressJS
- Typescript
- Prisma(PostgreSQL)

## Front-end

- Vite
- React
- Tailwind

## How to use

1. Clone the Repo

```
    git clone git@github.com:gabrielberlucci/fullstack-boilerplate.git
```

2. Delete .git

```
    rm .git
```

3. (Optional) Rename the root(app) folder

4. PNPM install

```
    cd server && pnpm i
    cd ../web && pnpm i
```
